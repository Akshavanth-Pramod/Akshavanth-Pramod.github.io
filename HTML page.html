What is HTML?

HTML (Hypertext Markup Language) is the standard language used to create web pages. It provides the structure of a webpage by defining elements like headings, paragraphs, images, and links.

Basic Structure of an HTML Document


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>HTML Basics</title>
    </head>
    <body>
        <h1>Welcome to HTML Basics!</h1>
        <p>This is your first HTML page.</p>
    </body>
</html>

HTML Tags

HTML uses tags to define different elements. Tags come in pairs: opening and closing tags. For example:

  • <h1> - defines a main heading.
  • <p> - defines a paragraph of text.
  • <a> - defines a hyperlink.
  • <img> - defines an image.

HTML Attributes

Attributes provide additional information about an element. They are always specified in the opening tag. Common attributes include:

  • href - specifies the URL for a link.
  • src - specifies the source file for an image or script.
  • alt - provides alternative text for an image.
  • id - specifies a unique identifier for an element.

Adding Links and Images

HTML allows you to add links and images to your webpage. Here's how:

Link Example:


<a href="https://www.example.com">Visit Example</a>

Image Example:


<img src="image.jpg" alt="Description of the image">

HTML Lists

Lists are used to group related content. HTML supports two types of lists:

  • Ordered List (<ol>) - displays items in a numbered format.
  • Unordered List (<ul>) - displays items with bullet points.

Example:


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>Step 1</li>
    <li>Step 2</li>
</ol>

here what the above code would look like

  • Item 1
  • Item 2
  1. Step 1
  2. Step 2

HTML Tables

HTML tables are used to display data in a tabular format. Here's an example:


<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>30</td>
    </tr>
</table>

This is how the above code would look like

Name Age
Alice 25
Bob 30

HTML Forms

HTML also provides forms to collect user input. Here's an example:


<form action="/submit" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name"><br>
    <input type="submit" value="Submit">
</form>

HTML Semantics

HTML5 introduces semantic elements that define the purpose of different parts of a webpage:

  • <header> - defines the header of a document or section.
  • <footer> - defines the footer of a document or section.
  • <article> - represents an independent piece of content.
  • <section> - defines a thematic grouping of content.
  • <nav> - represents navigation links.
  • <aside> - represents content aside from the main content.
