What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used to control the appearance of HTML elements on a webpage. It lets you apply styles such as colors, fonts, spacing, and layout to your webpage.

Basic CSS Syntax


/* This is a CSS comment */
selector {
    property: value;
}

CSS Selectors

CSS selectors are used to select the HTML elements you want to style. Here are some common selectors:

  • Element Selector: Selects all elements of a specific type (e.g., p for paragraphs).
  • Class Selector: Selects elements with a specific class (e.g., .button for elements with the class "button").
  • ID Selector: Selects an element with a specific ID (e.g., #header for the element with ID "header").
  • Group Selector: Applies styles to multiple selectors (e.g., h1, h2, p).

CSS Box Model

The CSS Box Model is a fundamental concept used to understand spacing and layout:

  • Content: The actual content inside the box (e.g., text, image).
  • Padding: Space between the content and the border.
  • Border: The edge surrounding the padding.
  • Margin: Space between the element and its neighbors.

Adding Styles to HTML

Here's how to add CSS to an HTML page:


<head>
    <link rel="stylesheet" href="styles.css">
</head>

Styling Images

CSS allows you to style images to make them visually appealing and fit your design:


img {
    max-width: 50%; /* Ensures the image is responsive */
    height: auto; /* Maintains aspect ratio */
    border: 2px solid #ccc; /* Adds a border around the image */
    border-radius: 10px; /* Rounds the corners of the image */
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2); /* Adds a subtle shadow */
}

Example: Styling a Button




button {
    background-color: #4CAF50; /* Green background */
    color: white; /* White text */
    padding: 10px 20px; /* Padding inside the button */
    border: none; /* No border */
    border-radius: 5px; /* Rounded corners */
    cursor: pointer; /* Pointer cursor on hover */
}

button:hover {
    background-color: #45a049; /* Darker green on hover */
}

CSS Layout Techniques

CSS provides various layout techniques for structuring a webpage:

  • Flexbox: A layout model for aligning items in a container.
  • Grid: A powerful layout system for creating two-dimensional designs.
  • Float: An older method for positioning elements.
  • Position: Specifies the positioning method for an element (e.g., relative, absolute, fixed).

Responsive Design with CSS

Responsive design ensures your webpage looks good on different devices. Key techniques include:

  • Media Queries: Apply different styles based on screen size.
  • Flexible Layouts: Use relative units like percentages and em/rem for widths.
  • Viewport Meta Tag: Ensures proper scaling on mobile devices.

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}
